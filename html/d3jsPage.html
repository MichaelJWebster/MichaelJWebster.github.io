Interactive Scatter Plot

The Gini coefficient is a measure of inequality within a country. A Gini coefficient of 1.0 would occur in a society were a single individual earn's all the income. A Gini coefficient of 0 would occur in a society where every individual has an exactly equal income.

The following interactive chart plots various population measures against the GINI coefficient, using data obtained from the OECD.

Mouse over any of the data circles to see the country the data is for.
